# MENSTORE Frontend

Modern e-commerce frontend for a menswear store built with **Next.js** and **Tailwind CSS**.

---

## 🚀 Overview

MENSTORE is a minimalist fashion e-commerce UI focused on clean design, smooth UX, and scalability.
This project represents the frontend part of a full-stack store (backend coming separately).

---

## ⚙ Tech Stack

* **Next.js (App Router)**
* **TypeScript**
* **Tailwind CSS**
* **React**

---

## ✨ Features

* Fullscreen hero section (image-first design)
* Sticky header with scroll behavior
* Responsive layout (mobile → desktop)
* Product grid with hover effects
* Clean and minimal UI inspired by modern fashion brands

## ▶ Getting Started

### 1. Clone the repository

```
git clone https://github.com/aslan8980/erkinsouls.git
cd erkinsouls/frontend
```

### 2. Install dependencies

```
npm install
```

### 3. Run development server

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```
